Q y A

¿Si entro como invitado(a) al aplicativo, puedo realizar un tramite?

Respuesta: No, al ingresar como invitado solo puedes visualizar la información correspondiente a cada uno de los trámites, sin embargo, recuerda que puedes registrarte en nuestro aplicativo y hacer uso de todas las funcionalidades

¿Si quiero realizar el tramite de un matrimonio, ¿Lo puedo hacer virtual?

Respuesta: No, estos tramites requieren de una presencialidad absoluta, estamos trabajando para que la mayoría de tramites se hagan por este medio y que la ceremonia sea presencial pero ya con todo el papeleo adelantado.

¿Si deseo desplazarme a la notaria física a realizar mis tramites, ¿Lo puedo hacer?

Respuesta: Si, para tu comodidad es que se ha diseñado este aplicativo, sin embargo uno de nuestros objetivos es reestructurar las notarias físicas para que sean mas optimas y cómodas para los usuarios y con un tiempo de espera menor a del siempre.

¿Puedo obtener un descuento en algún tramite?

Respuesta: No, las notarias no están autorizadas para hacer ningún tipo de descuento pero aun así los precios de cada uno de los tramites no son costosos y hay algunos que cuando es la primera vez son sin costos.

¿En mi perfil se guardan todos los tramites realizados?

Respuesta: Si, en el menú que hay debajo de la foto de perfil se encuentra una opción: "mi nube" que permite guardar los documentos ya pagos, sin embargo como algunos tramites tienen una fecha de vencimiento estos después del tiempo estipulado dejan de estar disponibles y los que no, continúan vigentes pero si deseas volver a descargar un documento debes volver a realizar el pago.

GUIA PARA ACCEDER AL APLICATIVO

Paso 1: Ingresa al aplicativo

Regístrate en unos pocos minutos de la forma que prefieras para almacenar tus tramites, o accede como invitado. Recuerda: al acceder como invitado no se te permite realizar ningún trámite solo visualizar la información de cada uno de ellos.

Paso 2: Selecciona tu tramite

Luego de completar el proceso de registro puedes acceder a cualquier tramite que necesites desde la pestaña del menú y seleccionarlo. En el tramite seleccionado se encuentra toda la información de este mismo y todos los documentos necesarios para validar. Recuerda: No puedes descargar el tramite hasta no haber realizado el pago correspondiente.

Paso 3: Lee requisitos del tramite

Luego de acceder al tramite que desear realizar debes leer cada uno de los requisitos que necesitas para hacer el proceso de validación. En muchas ocasiones dependiendo del tramite este puede tardar más en ser validado y por ende que este disponible para descargarlo, para esto en el proceso de inscripción se pide un correo donde se te notificará cuando el tramite este listo.

Paso 4: Llena el formulario

Luego de leer los requisitos puedes proceder a llenar el formulario que pide la información para realizar el tramite y si este es de entrega inmediata se te pondrá en una lista de espera para que el notario lo evalúe y proceda a habilitar la firma virtual. La lista de espera permite una mayor facilidad para atender las solicitudes en el orden correspondiente adicional a esto en tu perfil se encuentra una opción que muestra en que estado se encuentra el tramite.

Paso 5: Paga tu trámite

Paga tu trámite Para poder descargar tu trámite debes pagar primero el monto correspondiente, el aplicativo ofrece el pago de manera virtual y a través de los corresponsales y bancos. Nota: Cada notaria maneja una cuenta diferente, cada que realices un tramite al final te aparecerá el numero de la cuenta y donde puedes realizar el pago si no lo haces de manera virtual.
